Pour utiliser notre site:
-installer npm
-Mettre en place la BDD
-lancer le serveur web 
-Accéder au site

-Installer npm
1:depuis le dossier "projet-locamat" ouvrir un terminal et lancer la commande "npm install"

-Mettre en place la BDD
Il faut avoir un serveur HTTP et un serveur MySQL. Nous utilisons pour cela EasyPHP-Devserver-17.
2: ouvrir le dashboard de EasyPHP-Devserver. 
3: depuis le dashboard: démarrer le "HTTP SERVER" et le "DATABASE SERVER".
4: ouvrir le module "MySQL Administration : PhpMyAdmin 4.7.0"
5: Depuis phpMyAdmin: cliquer sur "Nouvelle base de données" puis sur "Importer".
6: cliquer sur "Choisir un fichier" et choisir "locamat_BDD_Script" qui se trouve dans le dossier "projet-locamat"

-lancer le serveur web
7: Dans le dossier "projet-locamat" lancer le fichier de commande "START"

-Accéder au site
8: Depuis un navigateur rentrer l'URL "localhost:3000"
9: pour vous connecter utilisez un des identifiants dans le fichier "usersLocamat.txt" qui se trouve dans le dossier "projet-locamat"
